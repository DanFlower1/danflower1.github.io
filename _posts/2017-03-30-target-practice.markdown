Instructions:

-Shoot all the targets within the given time

Gold = 4.00s

Silver = 6.00s

Bronze = 10.00s

Copper = 20.00s

---

Controls:

Left click = Fire

W = Walk forward

A = Walk left

S = Walk backwards

D = Walk right

Shift = Run

---
R = Change target colour to RED

G = Change target colour to GREEN

B = Change target colour to BLUE
